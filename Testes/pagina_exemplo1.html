Introdução e aparência do projeto

O projeto consiste em uma aplicação web para gerenciamento de clientes, que tem como objetivo permitir ao usuário a gestão de uma fábrica com produtos. A aplicação foi desenvolvida utilizando as tecnologias Nodejs, Express e React para criar um front-end do sistema. A aplicação possui diversas páginas que permitem: Cadastro de clientes, Cadastro de produtos e Consulta de produtos. Cadastrados clientes são registrados com os respectivos cadastros de produtos. A aplicação possui uma camada de autenticação para garantir a segurança dos dados do usuário.


Tecnologias utilizadas

  • ReactJS
  • NodeJS
  • Express
  • Axios
  • HTML5
  • CSS3
  • Bootstrap 4.1.2

Utilizado o framework Bootstrap

O framework foi usado para fazer as páginas do sistema. Ele é uma ferramenta com base em CSS e javascript que facilita a construção de layouts bem estruturados e com flexibilidade. Com isso, além de criar layouts direto de dentro do navegador, é possível gerenciar a página sem precisar de acesso via um servidor, já disponibilizando-lhe uma interface gráfica para alterações da página.
